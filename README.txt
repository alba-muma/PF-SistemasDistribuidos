PARA COMPILAR EL PROGRAMA SE DEBEN SEGUIR LOS SIGUIENTES PASOS:
1. Ejecutar el comando make en la terminal
2. Ejecutar el servidor en el puerto deseado con el comando ./servidor <puerto>
3. Ejecutar el servidor web de python con el comando python3 ./servidor_web.py
4. Ejecutar tantos clientes como se desee con el comando python3 ./client.py -s <ip> -p <puerto>
5. Se abrirá una interfaz en la que se podrá probar el programa.
